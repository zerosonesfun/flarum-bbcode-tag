# Flarum BBCode Tag

Adds a [t]keyword[/t] BBCode to Flarum. Turns wrapped text into a search link.

## Install & Update
`composer require zerosonesfun/flarum-bbcode-tag`

`php flarum cache:clear`

## Remove
`composer remove zerosonesfun/flarum-bbcode-tag`
